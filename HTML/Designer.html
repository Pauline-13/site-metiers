UI Designer Game


Appellations possibles

  • Game UI Artist (Très courant dans les studios européens et indés)
  • HUD Artist (Spécialisé dans les têtes d'affichage : barres de vie, score...)
  • Visual Designer Gaming (Plutôt dans les gros studios)
  • UI Visual Designer (Très axé esthétique, souvent dans le mobile)

Missions

  • Créer les éléments visuels de l’interface : boutons, icônes, HUD, menus, etc.
  • Adapter le style graphique à l’univers du jeu
  • Collaborer avec les game designers, UX designers et développeurs
  • Prototyper l’interface (wireframes, maquettes, mockups)
  • Réaliser des guides de style visuel

Compétences nécessaires

  • Graphisme : Photoshop, Illustrator, Figma ou XD
  • Sens visuel : hiérarchie, couleurs, typographie, lisibilité
  • Adaptabilité graphique selon le style du jeu
  • Communication avec l’équipe design/dev
  • Connaissances techniques (moteurs, performances...)

Outils utilisés

  • Création : Photoshop, Illustrator
  • Prototypage : Figma, Adobe XD
  • Moteurs : Unity, Unreal
  • Animations UI : After Effects, Spine

Salaire moyen

  • Junior : 30 000€ – 35 000€ brut/an
  • Confirmé : 35 000€ – 40 000€ brut/an
  • Senior / Lead : jusqu’à 60 000€ brut/an

Et la suite ?

À court terme :

  • Approfondir mes connaissances en UI de jeu vidéo
  • Créer un portfolio spécialisé (HUD, menus, interfaces immersives...)
  • Apprendre les outils (Photoshop, Illustrator, After Effects...)
  • Expérimenter sur des petits projets / prototypes
  • Rechercher un stage dans un studio

À moyen terme :

  • Travailler dans un studio de jeu
  • Maitriser Unity ou Unreal pour l’intégration UI
  • Optimiser la lisibilité et l’esthétique sur divers projets
  • Participer à des Game Jams

À long terme :

  • Superviser la direction visuelle d’un jeu
  • Participer à des conférences ou salons du jeu vidéo
  • Former de futurs UI Designers Game
